    Understanding Blood Types

    Learn about blood type compatibility. Every type is needed, but some are rarer than others.

    A

    Can Donate To: A, AB

    Can Receive From: A, O

    B

    Can Donate To: B, AB

    Can Receive From: B, O

    AB

    Can Donate To: AB

    Can Receive From: All Types (Universal Recipient)

    O

    Can Donate To: All Types (Universal Donor)

    Can Receive From: O
